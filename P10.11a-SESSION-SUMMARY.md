## Patterns Documented

### Pattern 1: Structural Recursion on Fuel

```lean
theorem loop_preserves_tokens (s : ScannerState) (acc : α) (fuel : Nat) :
    (loop s acc fuel).snd.tokens = s.tokens := by
  induction fuel generalizing s acc with
  | zero => unfold loop; rfl
  | succ fuel' ih =>
    unfold loop
    split
    · split
      · rw [ih]  -- Apply IH to modified state
        exact advance_preserves_tokens s  -- Bridge back
      · rfl
    · rfl
```

**Used in:** `collectAnchorNameLoop_preserves_tokens` ✅

**To apply to:** 3 remaining collect loop helpers

### Pattern 2: Monadic Proof with Helper

```lean
theorem scan*_adds_token (s : ScannerState) (s' : ScannerState)
    (h : scan* s = .ok s') :
    s'.tokens.size = s.tokens.size + 1 := by
  unfold scan* at h
  simp only [bind, Except.bind] at h
  split at h <;> try contradiction
  rename_i heq  -- Extract hypothesis from split
  have h_collect := collect*Loop_preserves_tokens ...args... heq
  have h_adv := advance_preserves_tokens s
  injection h with h_eq; subst h_eq
  rw [emitAt_tokens_size, h_collect, h_adv]
```

**Used in:** All 3 scalar scanning lemmas ✅

**To apply to:** scanBlockScalar (more complex due to header parsing)

### Pattern 3: Composition Without Monads

```lean
theorem scan*_adds_token (s : ScannerState) :
    (scan* s).tokens.size = s.tokens.size + 1 := by
  unfold scan*
  have h1 := helper1_preserves ...
  have h2 := helper2_preserves ...
  rw [emitAt_tokens_size, h1, h2]
```

**Used in:** `scanAnchorOrAlias_adds_one_token` ✅
