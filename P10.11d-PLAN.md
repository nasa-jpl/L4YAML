## Technical Challenges

### Challenge 1: Imperative Loop Reasoning
The `scan` function uses `for _ in [:fuel * 4] do`, which desugars to complex recursion in Lean.

**Structure of scan** (Scanner.lean:1968-1988):
```lean
def scan (input : String) := do
  let mut s := ScannerState.mk' input
  s := s.emit .streamStart              -- Line 1970
  -- Handle BOM (optional)
  for _ in [:fuel * 4] do               -- Line 1976
    match ← scanNextToken s with
    | some s' => s := s'
    | none =>
      -- Error checks, then:
      let final := unwindIndents s (-1)
      let final := final.emit .streamEnd -- Line 1987
      return final.tokens                -- Line 1988
  .error (.fuelExhausted s.line s.col)
```

**Key insight**: `emit` is just `Array.push`:
```lean
def ScannerState.emit (s : ScannerState) (tok : YamlToken) : ScannerState :=
  { s with tokens := s.tokens.push { pos := s.currentPos, val := tok } }
```
