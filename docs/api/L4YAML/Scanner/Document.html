    Scanner — Document Boundaries and Directives #

    Document-start / document-end markers (---, ...) and YAML directives (%YAML, %TAG, reserved) — everything at §6.8 and §9.1.2 of the spec.

    Split from Scanner.lean during Blueprint Initiative 1 Phase 2.

    Scope #

    • Boundary detection: atDocumentStart, atDocumentEnd, atDocumentBoundary.
    • Directive parsing helpers: collectDirectiveNameLoop, collectVersionMajorLoop, collectVersionMinorLoop, collectTagHandleDirectiveLoop, collectTagPrefixLoop.
    • Directive scanners: scanYamlDirective, scanTagDirective, scanDirective.
    • Marker scanners: scanDocumentStart, scanDocumentEnd, skipDocEndWhitespace.
