    def L4YAML.Config.configParserLimits :
    ParserLimits

    Hardcoded limits for config parsing — config YAML is small and trusted but we still enforce basic safety. These are tighter than .strict since config documents should never be large or complex.

      Except String α

      Parse a YAML string into a typed value using safe parsing with tight config-specific limits. This is the bootstrapping entry point: the parser parses its own configuration.

      Combines parseYamlSingleSafe (with configParserLimits) and Schema.fromYaml? into a single safe pipeline.
