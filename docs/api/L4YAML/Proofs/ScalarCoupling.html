    Scalar Collection Coupling #

    Coupling theorems for scalar content collection functions, connecting the scanner's scalar-scanning loops to the formal surface syntax.

    Each scalar scanning function preserves ScannerSurfCorr on every .ok return path. Content-string accumulation and token emission are irrelevant to correspondence since they only affect tokens/simpleKey/comments fields, none of which appear in ScannerSurfCorr.
