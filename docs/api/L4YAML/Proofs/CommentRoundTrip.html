    Comment Round-Trip Proofs (Phase G6) #

    This module proves properties of the comment-aware emitter emitWithComments and its interaction with parseYamlWithComments.

    Key Results #

    §1: Emitter Structural Properties #

    The comment-aware emitter produces well-formed output that starts with comment lines (if any) followed by the canonical value emission.

    §2: Comment Text Round-Trip #guard Checks #

    Compile-time verification that comment texts survive the emitWithComments → parseYamlWithComments round-trip for concrete cases.

    §3: Value Round-Trip Through Comment Emission #

    The value tree is preserved through comment-aware emission, since emitWithComments uses the canonical emitter for the value portion.

    Strategy #

    Following the approach of Proofs/RoundTrip.lean:

    • Concrete cases: #guard compile-time checks for emit→parse→commentTexts
    • native_decide: Concrete structural theorem proofs
    • Algebraic: Properties of emitWithComments composition

    The full universal comment round-trip theorem requires unfolding through both the scanner and parser (~10K lines combined). We verify it via #guard for many concrete cases and prove the algebraic infrastructure that enables the universal statement.

    Zero Axioms #

    All theorems are machine-checked. No sorry, no axiom, no partial.

    §1: Emitter Structural Properties #

    The comment-aware emitter is a composition of comment-line emission and canonical value emission. These properties verify the structure.

    theorem L4YAML.Proofs.CommentRoundTrip.emitWithComments_empty_comments :
    Emit.emitWithComments { value := YamlValue.plainScalar "hello" } = Emit.emit (YamlValue.plainScalar "hello")

    emitWithComments on a document with no comments equals emit on the value.

    theorem L4YAML.Proofs.CommentRoundTrip.emitWithComments_one_comment :
    Emit.emitWithComments { value := YamlValue.plainScalar "hello", comments := #[({ offset := 0, line := 0, col := 0 }, { text := " a comment", position := CommentPosition.inline })] } = "# a comment\n\"hello\""

    emitWithComments on a document with one comment produces a comment line followed by the value.

    theorem L4YAML.Proofs.CommentRoundTrip.emitCommentLines_empty :
    Emit.emitCommentLines #[] = ""

    emitCommentLines on empty comments produces empty string.

    emitCommentLines on a single comment produces #text\n.
