    Structure, Document & Directive Coupling #

    Coupling theorems for structural scanning: flow/block indicators, node properties (anchors + tags), indentation management, document boundaries, and directives.

    Each function preserves ScannerSurfCorr on every .ok return path. Most field updates (tokens, flowLevel, indents, flowStack, simpleKey, simpleKeyStack, simpleKeyAllowed, explicitKeyLine, allowDirectives, seenYamlDirective, directivesPresent, documentEverStarted, definedAnchors) are irrelevant to correspondence since they only affect fields outside ScannerSurfCorr's 4-field window (input, offset, col, inputEnd).
