    Composition Theorems #

    This module provides composition theorems for the tokenized YAML pipeline:

      String ──Scanner.scan──→ Array (Positioned YamlToken) ──parseStream──→ Array YamlDocument
         └── parseYamlRaw ──────────────────────────────────────────────┘
         └── parseYaml ────────("compose" aliases)────────────────────────→ Array YamlDocument

    Architecture #

    P10.5 rewrite: The previous version composed old char-level parser specs (position algebra, fuel wrappers, lean4-parser combinator extensions) for the pre-P10 yamlStream parser. Now that the public API delegates to TokenParser.parseYamlRaw / TokenParser.parseYaml, the relevant composition is about the Scanner → TokenParser pipeline.

    Sections #

    1. Scanner–TokenParser Pipeline (§1): parseYamlRaw decomposes into Scanner.scan followed by parseStream. Error propagation flows naturally from do-notation.

    2. Compose Layer (§2): parseYaml applies YamlDocument.compose to each document from parseYamlRaw, resolving aliases and stripping anchors. The key theorem parseYaml_ok_iff (in Completeness.lean) is re-exported here for convenience.

    §1 Scanner–TokenParser Pipeline #

    parseYamlRaw is do let tokens ← Scanner.scanFiltered input; parseStream tokens. The following theorems decompose this pipeline.

    source
    theorem L4YAML.Proofs.Composition.parseYamlRaw_pipeline (input : String) (tokens : Array (Positioned YamlToken)) (docs : Array YamlDocument) (h_scan : Scanner.scanFiltered input = Except.ok tokens) (h_parse : TokenParser.parseStream tokens = Except.ok docs) :
    TokenParser.parseYamlRaw input = Except.ok docs

    parseYamlRaw decomposes into scanning then parsing: if both stages succeed, the result is the parseStream output on the scanned tokens.

    source
    theorem L4YAML.Proofs.Composition.parseYamlRaw_ok_decompose (input : String) (docs : Array YamlDocument) (h : TokenParser.parseYamlRaw input = Except.ok docs) :
    ∃ (tokens : Array (Positioned YamlToken)), Scanner.scanFiltered input = Except.ok tokens ∧ TokenParser.parseStream tokens = Except.ok docs

    If parseYamlRaw succeeds, then both Scanner.scan and parseStream must have succeeded.

    source
    theorem L4YAML.Proofs.Composition.parseYamlRaw_scan_error (input : String) (e : ScanError) (h : Scanner.scanFiltered input = Except.error e) :
    TokenParser.parseYamlRaw input = Except.error e

    If Scanner.scan fails, parseYamlRaw fails with the same error.

    source
    theorem L4YAML.Proofs.Composition.parseYamlRaw_parse_error (input : String) (e : ScanError) (tokens : Array (Positioned YamlToken)) (h_scan : Scanner.scanFiltered input = Except.ok tokens) (h_parse : TokenParser.parseStream tokens = Except.error e) :
    TokenParser.parseYamlRaw input = Except.error e

    If Scanner.scan succeeds but parseStream fails, parseYamlRaw fails with the same parse error.

    §2 Compose Layer #

    parseYaml extends parseYamlRaw by applying YamlDocument.compose to each raw document, resolving aliases and stripping anchor annotations. The primary decomposition theorem parseYaml_ok_iff is in Completeness.lean; here we provide additional convenience forms.

    source
    theorem L4YAML.Proofs.Composition.parseYaml_of_parseYamlRaw_ok (input : String) (docs : Array YamlDocument) (h : TokenParser.parseYamlRaw input = Except.ok docs) :
    TokenParser.parseYaml input = Except.ok (Array.map YamlDocument.compose docs)

    If parseYamlRaw succeeds, parseYaml succeeds with composed documents.

    source
    theorem L4YAML.Proofs.Composition.parseYaml_of_parseYamlRaw_error (input : String) (e : ScanError) (h : TokenParser.parseYamlRaw input = Except.error e) :
    TokenParser.parseYaml input = Except.error e

    If parseYamlRaw fails, parseYaml fails with the same error.

    source
    theorem L4YAML.Proofs.Composition.parseYaml_pipeline (input : String) (tokens : Array (Positioned YamlToken)) (docs : Array YamlDocument) (h_scan : Scanner.scanFiltered input = Except.ok tokens) (h_parse : TokenParser.parseStream tokens = Except.ok docs) :
    TokenParser.parseYaml input = Except.ok (Array.map YamlDocument.compose docs)

    Full pipeline composition: Scanner.scan → parseStream → compose. If scanning and parsing both succeed, parseYaml returns composed documents.
