    Surface Syntax Coupling Proofs #

    Theorems connecting the parser implementation to the formal surface syntax predicates. These form the proof backbone for the acceptance strictness property: parseYaml s = .ok docs → InYamlLanguage s.

    Architecture #

    The coupling works through a state correspondence relation that bridges the Scanner's ScannerState (byte-level offsets into a String) with the Surface syntax's SurfPos (character-level List Char + column).

    At each scanner step, we show that the scanner's state change corresponds to a surface syntax production step:

    • skipSpaces n → SIndent n
    • consumeNewline → SBBreak
    • skipWhitespace → SSWhite*
    • skipToContent → SSLComments

    Strategy #

    Bottom-up: prove character-level coupling first, then compose into production-level coupling, then chain through the full pipeline.

    §1 Character-Level Lemmas #

    These lemmas establish basic properties of the surface syntax combinators that are used throughout the coupling proofs.

    source
    theorem L4YAML.Proofs.SurfaceCoupling.SIndent_zero (s : Surface.SurfPos) :

    SIndent 0 is the identity (no characters consumed).

    source
    theorem L4YAML.Proofs.SurfaceCoupling.SIndent_succ (n : Nat) (rest : List Char) (col : Nat) (s' : Surface.SurfPos) (h : Surface.SIndent n { chars := rest, col := col + 1 } s') :
    Surface.SIndent (n + 1) { chars := ' ' :: rest, col := col } s'

    SIndent (n+1) requires a leading space.

    source
    theorem L4YAML.Proofs.SurfaceCoupling.SIndent_col (n : Nat) (s s' : Surface.SurfPos) (h : Surface.SIndent n s s') :
    s'.col = s.col + n

    SIndent n advances column by exactly n.

    theorem L4YAML.Proofs.SurfaceCoupling.SIndent_chars (n : Nat) (s s' : Surface.SurfPos) (h : Surface.SIndent n s s') :
    s'.chars = List.drop n s.chars

    These lemmas show how surface syntax productions compose.

    §3 Indent Coupling (Scanner → Surface) #

    The key theorem: scanner's skipSpaces consuming n spaces corresponds to SIndent n in the surface syntax.

    source
    theorem L4YAML.Proofs.SurfaceCoupling.spaces_give_SIndent (n : Nat) (chars : List Char) (col : Nat) (hpre : List.take n chars = List.replicate n ' ') (hlen : chars.length ≥ n) :
    Surface.SIndent n { chars := chars, col := col } { chars := List.drop n chars, col := col + n }

    If a list starts with n spaces, SIndent n holds on the corresponding SurfPos, advancing by exactly n characters.

    §4 Line Break Coupling (Scanner → Surface) #

    The scanner's consumeNewline consuming CR/LF/CRLF corresponds to SBBreak in the surface syntax.

    source
    theorem L4YAML.Proofs.SurfaceCoupling.lf_gives_SBBreak (rest : List Char) (col : Nat) :
    Surface.SBBreak { chars := '\n' :: rest, col := col } { chars := rest, col := 0 }

    If the head character is \n, SBBreak holds.

    source
    theorem L4YAML.Proofs.SurfaceCoupling.crlf_gives_SBBreak (rest : List Char) (col : Nat) :
    Surface.SBBreak { chars := '\x0d' :: '\n' :: rest, col := col } { chars := rest, col := 0 }

    If the head characters are \r\n, SBBreak holds.

    source
    theorem L4YAML.Proofs.SurfaceCoupling.cr_gives_SBBreak (rest : List Char) (col : Nat) :
    Surface.SBBreak { chars := '\x0d' :: rest, col := col } { chars := rest, col := 0 }

    If the head character is \r (not followed by \n), SBBreak holds.
