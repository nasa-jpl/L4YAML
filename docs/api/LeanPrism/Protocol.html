                                      inductive LeanPrism.ServerToClientMessage :
                                      Type

                                      Messages sent from the lean-prism server to the lean-tui client. Each message is a single JSON object terminated by a newline.

                                      • connected (serverMode : Option ServerOperatingMode) (fileUri : Option String) (port portRangeMin portRangeMax : Option Int) : ServerToClientMessage

                                        Sent when the TUI client first connects to the lean-prism server. Provides information about the server's operating mode, the port this worker is using, and the file it is handling.

                                      • cursor (uri : String) (position : LineCharacterPosition) (method : String) : ServerToClientMessage

                                        Sent when the editor cursor moves to a new position. Allows the TUI to track cursor location even when no proof DAG is available.

                                      • error (error : String) : ServerToClientMessage

                                        Sent when an error occurs during proof DAG extraction or other server operations. The TUI should display the error message to the user.

                                      • dag (uri : String) (position : LineCharacterPosition) (dag : Option GenericDag) : ServerToClientMessage

                                        Unified DAG message - dispatches to appropriate TUI mode based on display_style. Can represent proof trees, data flow graphs, or effect flow graphs.

                                      • fileFocused (uri : String) (focused : Bool) : ServerToClientMessage

                                        Sent when the editor focuses a file (e.g., via textDocument/didOpen or tab switch). Allows the TUI to immediately follow file switches without waiting for cursor movement.
