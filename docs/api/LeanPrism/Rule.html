    structure LeanPrism.Rule (Result : Type) :
    Type

    A generic rule for analyzing expressions. Rules are checked in priority order (highest first) and the first matching rule is applied.

    • name : Lean.Name
    • priority : Nat
    • appliesTo : Lean.Expr → Bool
    • run : Lean.Expr → Lean.LocalContext → Option Lean.Syntax → Lean.MetaM (Option Result)
