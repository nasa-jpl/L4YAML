                  def LeanPrism.Generator.classifyTree (tree : Lean.Elab.InfoTree) (fileMap : Lean.FileMap) (position : Lean.Lsp.Position) :
                  ClassificationResult

                  Position-aware classification of an InfoTree. Returns both the classification and the reason for diagnostics. Priority chain (highest to lowest confidence):

                  1. TacticInfo at cursor position
                  2. Block kind (theorem/lemma/example)
                  3. Effectful terms at position
                  4. Definition kind (def/abbrev default to dataFlow)
                  5. Tree-wide term mode heuristic
