### Phase 5: Option A Phase 1 — Refactoring

**Goal**: Transform imperative loops to structural recursion

#### Implementation Changes

**Before** (imperative):
```lean
def scan (input : String) := do
  let mut s := ScannerState.mk' input
  s := s.emit .streamStart
  for _ in [:fuel * 4] do                -- ❌ Imperative loop
    match ← scanNextToken s with
    | some s' => s := s'                 // ❌ Mutable state
    | none => return final.tokens        -- ❌ Early return
  .error (.fuelExhausted ...)
```

**After** (structural recursion):
```lean
def scanLoop (s : ScannerState) (fuel : Nat) :=
  match fuel with                        -- ✅ Structural recursion
  | 0 => .error (.fuelExhausted ...)
  | fuel' + 1 =>
    match scanNextToken s with
    | .ok none => .ok (final.emit .streamEnd).tokens
    | .ok (some s') => scanLoop s' fuel'  -- ✅ Recursive call
termination_by fuel                      -- ✅ Termination proven

def scan (input : String) :=             -- ✅ Pure function
  let s := (ScannerState.mk' input).emit .streamStart
  scanLoop s ((input.utf8ByteSize + 1) * 4)
```
