## Technical Details

### skipToContentLoop Structure

```lean
def skipToContentLoop (s : ScannerState) (fuel : Nat) : Except ScanError ScannerState :=
  match fuel with
  | 0 => .ok s
  | fuel' + 1 => do
    let mut s' := s
    -- ... whitespace/comment/newline handling ...
    match s'.peek? with
    | some c =>
      if isLineBreak c then
        s' := consumeNewline s'
        if !s'.isInFlowSequence then
          skipToContentLoop { s' with simpleKeyAllowed := true } fuel'
        else
          skipToContentLoop s' fuel'
      else .ok s'
    | none => .ok s'
termination_by fuel
```

**Key properties**:
1. Structurally recursive on `fuel` parameter
2. Base case: returns input state unchanged
3. Recursive case: processes one line, recurses with `fuel' = fuel - 1`
4. Termination proof automatic via `termination_by fuel`

**Why this enables proofs**:
- Induction on fuel provides natural proof structure
- No imperative mutation to reason about
- Each recursive call is clearly smaller (fuel decreases)
