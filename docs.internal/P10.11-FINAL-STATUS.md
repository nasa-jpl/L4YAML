### Lesson 1: Imperative Code Blocks Verification

**Finding**: for-loops desugar to opaque ForIn operations
- Immediate failure on all proof tactics
- No visibility into iteration structure
- **Solution**: Refactor to structural recursion

### Lesson 2: Test-Driven Refactoring Works

**Finding**: Comprehensive tests enable bold refactoring
- 113/113 tests provide immediate validation
- Zero regressions after both refactorings
- **Confidence**: Functional equivalence maintained

### Lesson 3: Helper Lemmas Accumulate Value

**Finding**: Early lemmas reused in later proofs
- `emit_preserves_tokens_at`: Used 5+ times
- `unwindIndents_preserves_prefix`: Critical for scanLoop
- **Strategy**: Invest in reusable infrastructure

### Lesson 4: Dependent Types Need Care

**Finding**: Array indexing with Fin types is tricky
- Explicit bounds needed everywhere
- `simp only` works better than `rw` for dependent rewrites
- **Pattern**: Establish bounds early, use calc chains

**Finding**: P10.11d estimate vs. actual
- **Estimated**: 7-12 days for Option A
- **Actual**: 7 days to 60% completion
