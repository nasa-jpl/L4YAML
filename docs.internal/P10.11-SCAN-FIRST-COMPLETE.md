## Proof Techniques Used

### Technique 1: Array Index Management

**Challenge**: Dependent typing with array indices
- `tokens[i]'(h : i < tokens.size)` requires explicit proofs

**Solution**: Establish size bounds early
```lean
have h_i_lt_unwind : i < s_unwind.tokens.size := by omega
have h_i_lt_emitted : i < (s_unwind.emit .streamEnd).tokens.size := by
  rw [h_emit_size]; omega
```

### Technique 2: Equality Propagation

**Challenge**: Rewriting with dependent types fails
- `rw` struggles with array equalities in dependent contexts

**Solution**: Use `simp only` with specific hypotheses
```lean

```
