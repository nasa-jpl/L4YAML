### 2026-03-05 Update: Axiom Eliminated ✅

The single axiom `skipToContent_preserves_tokens` has been **fully proven**.

**Root cause**: The original `skipToContentLoop` used `do`+`let mut` notation,
which Lean 4 desugars into nested `__do_jp` join points. After `unfold`, `split`
cannot decompose these synthetic constructs.

**Solution**: Refactored `skipToContentLoop` into three pure functions:
- `skipToContentWs` — explicit `if`/`match`/`let` for whitespace/indentation
- `skipToContentComment` — pure function for comment skipping
- `skipToContentLoop` — clean `match`-based recursion, no `do`/`mut`

**Key tactic insight**: After `unfold`, `split at h` fails when the hypothesis
contains `have x := ...; ...` (let-bindings). Inserting `simp only [] at h`
between `split` calls beta-reduces these bindings, making subsequent `split`
calls succeed.

**Result**: 4 new theorems, 0 axioms, 0 sorry, 191/191 build.

### 3. System Limitation Identified and Resolved ✅

**Discovery**: `skipToContentLoop` not accessible to `unfold`+`split` proof strategy

**Root cause** (confirmed 2026-03-05): `do`+`let mut` desugars into nested

`do`/`mut` — use explicit `if`/`match`/`let` chains instead.
